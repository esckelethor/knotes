> **PUBLISHED WEBAPP: [K-NOTES ~ K-주석](https://esckelethor.github.io/knotes/)** *(webapp under construction)*

Initially, a simple webapp was created for studying Korean by collecting notes taken during study sessions

After some time, we decided to add more features, such as:
- Include Spotify with dynamic playlists
- Include a user manual
- Include exercises by topic
- Include exams with timer and score
- Create flashcards for vocabulary
- Include multiple languages:
  - **KR** *(Status: Under construction)*
  - **EN** *(Status: Under construction)*
  - **FR** *(Status: Pending)*
