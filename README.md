# K-NOTES ~ K-주석

## ES
Al principio se creo una web simple para estudiar coreano mediante la recopilación de notas tomadas mientras se estudiaba.

Pasado un tiempo hemos decidido darle más funcionalidades, como:
 - Incluir Spotify con listas dinamicas
 - Incluir un manual de uso
 - Incluir multiidioma:
   
     | Idioma | Progreso |
     |---|---|
     | Coreano | ⭕ En construcción |
     | Inglés | ⭕ En construcción |
     | Francés | ❌ Pendiente | 
     | Japonés | ❌ Pendiente |
   
 - Incluir ejercicios por tema
 - Incluir examenes con temporizador y puntuación
 - Creación de tarjetas para estudiar el vocabulario

### Webapp publicada (instalable como un tab a pantalla completa en Android)
https://esckelethor.github.io/knotes/ <i>(webapp en construcción)</i>

### Desarrollado con
 - HTML5
 - CSS3
 - Vanilla JS (ECMAScript 15)
 - Framework personalizado vQuery replicando funcionalidades de jQuery

## EN
Initially, a simple web was created for studying Korean by compiling notes taken during study sessions.

After some time, we decided to add more features, such as:
- Including Spotify with dynamic playlists
- Including a user manual
- Including multiple languages:

    | Language | Progress |
    |---|---|
    | Korean | ⭕ Under construction |
    | English | ⭕ Under construction |
    | French | ❌ Pending |
    | Japanese | ❌ Pending |

- Include exercises by topic
- Include exams with timer and score
- Create flashcards for vocabulary
