Verbos: presente

Presente simple

Forma

Para I, you, we, they utilizamos el infinitivo, para he, she, it se añade s al final del verbo, ej:
I, you, we, they talk, eat, learn, do, go...
he, she, it talks, eats, learns, does, goes...

Estructura

Afirmativo: Sujeto + verbo
Ej: I talk -> Yo hablo / She eats -> Ella come
Negativo: Sujeto + verbo auxiliar (to do) + auxiliar negativo (not) + verbo
Ej: I don't talk -> Yo no hablo / She doesn't eat -> Ella no come
Interrogativo: Verbo auxiliar (to do) + sujeto + verbo principal + ?
Ej: Do you talk? -> ¿Hablas tu? / Does she eat? -> ¿Come ella?

Usos

1. Para hablar de cosas que suceden habitualmente, acompañados por adverbios de tiempo
always -> siempre
every day -> cada día
usually -> normalmente
often -> a menudo
sometimes -> a veces
rarely -> raramente
hardly ever -> casi nunca
never -> nunca
Ej: She never eats vegetables -> Ella nunca come las verduras
*Excepcion: Cuando se usa to be el verbo va delante del adverbio
Ej: You are rarely late -> Tu raramente llegas tarde

2. Para hablar de cosas generales
Ej: She works in a hospital -> Ella trabaja en una hospital

3. Para eventos programados en un futuro proximo
Ej: The party is tonight -> La fiesta es esta noche

4. Con el imperativo
Ej: Don't cry! -> No llores!

Presente continuo

Forma

Se utiliza el verbo to be + verbo acabado en ing
I am talking, eating, learning, doing, going...
you, we, theyare talking , eating, learning, doing, going...
he, she, it is talking, eating, learning, doing, going...

Estructura

Afirmativo: Sujeto + verbo auxiliar (to be) + verbo+ing
Ej: I'm talking -> Yo estoy hablando
Negativo: Sujeto + verbo auxiliar (to be) + auxiliar negativo (not) + verbo+ing
Ej: I'm not talking -> Yo no estoy hablando
Interrogativo: Verbo auxiliar (to be) + sujeto + verbo+ing + ?
Ej: Are you talking? -> ¿Estás tu hablando?

Usos

1. Para hablar sobre algo que está pasando en el momento en el que hablamos
Ej: I'm studying now -> Yo estoy estudiando ahora

2. Para hablar de algo que está sucediendo en la actualidad pero no necesariamente cuando hablamos, se utiliza junto a currently, lately, these days
Ej: They're learning English -> Ellos están aprendiendo inglés

3. Para hablar de algo que está ya decidido que se hará en el futuro próximo
Ej: I'm going to the party tonight -> Yo voy a la fiesta esta noche
*Excepcion: hay verbos con los que no se suele utilizar el presente continuo
be -> ser/estar
want -> querer
need -> necesitar
know -> saber/conocer
prefer -> preferir
remember -> recordar
understand -> comprender
care -> cuidar
see -> ver
hear -> oír
smell -> oler
believe -> creer
belong -> pertenecer
cost -> costar
seem -> parecer
exist -> existir
own -> poseer
like -> gustar
dislike -> desagradar
love -> amar
hate -> odiar
fear -> temer
envy -> envidiar
mind -> importar
Ej: He needs a new bike -> El necesita una bicicleta nueva

Presente continuo de los verbos

Con los verbos estáticos y no de actividad no se utiliza el presente continuo
1. Verbos de sentido
feel -> sentir
hear -> oir
see -> ver
smell -> oler
taste -> sabor
Ej involuntaria: I can see you -> Te veo
Ej voluntaria: I am seeing you later -> Te veré luego
*Si la percepción es voluntaria, se puede usar la forma continua

2. Verbos emotivos
believe -> creer
dislike -> no gustar
like -> gustar
realize -> dar cuenta
recognize -> reconocer
want -> querer
doubt -> dudar
imagine -> imaginar
love -> encantar, amar
remember -> recordar, acordarse
suppose -> suponer
wish -> esperar
hate -> odiar
know -> saber, conocer
prefer -> preferir
think -> creer
understand -> entender, comprender
Ej: I don't believe you! -> ¡No te creo!

3. Verbos de estado o verbos abstractos
be -> ser, estar
cost -> costar
exist -> existir
seem -> parecer
need -> necesitar
care -> importar
contain -> contener
Ej: She is ill -> Ella está enferma

4. Verbos de posesión
belong -> pertenecer
own -> poseer
have -> tener
possess -> poseer
Ej: That belongs to me -> Eso me pertenece

5. Verbos de comunicación
agree -> acordar
astonish -> asombrar
deny -> negar
disagree -> no estar de acuerdo
impress -> impresionar
mean -> significar
please -> agradar
promise -> prometer
satisfy -> satisfacer
surprise -> sorprender
Ej: I disagree -> Yo no estoy de acuerdo
