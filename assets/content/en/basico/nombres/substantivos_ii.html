Substantivos

Substantivos contables e incontables

Substantivos contables

Son aquellos que se pueden contar, ej: one cat -> un gato

Substantivos incontables

Son aquellos que no se pueden contar de forma individual porque forman parte de un grupo, estos, son tratados como singular

salt -> sal
wood -> madera
tea -> té
wine -> vino
sugar -> azúcar
bread -> pan
furniture -> muebles
hair -> pelo
information -> información
money -> dinero
weather -> tiempo
time -> tiempo
rice -> arroz

Para individualizarlos deben llevar alguna palabra que los divida como gram, piece, cup, ej:

a gram of salt -> un gramo de sal
two cups of tea -> dos tazas de té

Reglas gramaticales

1. Substantivos contables tienen forma plural, ej: dress -> dresses (vestidos)

2. Substantivos incontables no tienen forma plural, ej: rice (arroz)

3. se puede usar a, an con Substantivos contables en singular, ej: an apple -> una manzana

4. No podemos usar a, an con Substantivos incontables, ej: milk (leche)

5. se puden usar números delante de un contable, ej: 3 apples -> 3 manzanas

6. No podemos usar números delante de un incontable, ej: rice -> arroz

Haber

Para indicar la existencia de algo se usa to be, conjugamos este verbo si el nombre que va después está en plural o en singular y si es contable o incontable

There is

Su forma corta es there's, con Substantivos contables en singular y con incontables

Ej contable: There is an apple -> hay una manzana

Ej incontable: There is milk -> hay leche

There are

Sólo con Substantivos contables en plural, ej: There are five pencils -> hay 5 lapices

Nota: No se conjuga con el futuro, el pasado perfecto y con los verbos modales

There vs It

There se utiliza cuando hablamos de algo por primera vez, para indicar que existe. It se utiliza para hablar de una cosa especifica que ya fue mencionada

Ej: There is a pen on the table. It is my pen -> Hay un bolígrafo en la mesa. Es mi bolígrafo

Cuantificadores

Los cuantificadores indican la cantidad de un nombre, responden a la pregunta ¿cuantos?. Siempre van delante del nombre

Many: muchos/muchas - contable

Uso: se utiliza para expresar gran cantidad. Se usa principalmente en frases negativas e interrogativas con contables en plural, en las afirmativas, normalmente se utiliza a lot

Ej frase negativa: There are not many flowers -> No hay muchas flores

Ej frase interrogativa: How many flowers do you have? -> ¿Tienes muchas flores?

Ej frase afirmativa: I have many flowers -> Yo tengo muchas flores

*Nota: Se puede combinar con too -> demasiados o con so -> tantos para expresar algo excesivo, ej:

There are too many changes for me -> Hay demasiados cambios para mi

There are so many things I want to do -> Hay tantas cosas que deseo hacer

Much: mucho/muchas - incontable

Uso: se utiliza para expresar gran cantidad. Se usa principalmente en frases negativas e interrogativas con incontables en singular

Ej frase negativa: There is not much rice -> No hay mucho arroz

Ej frase interrogativa: How much rice does he have? -> ¿Tienes mucho arroz?

Ej frase afirmativa: We have much to do -> Tenemos mucho que hacer

*Nota: Se puede combinar con too -> demasiados o con so -> tantos para expresar algo excesivo, ej:

There is too much work -> Hay demasiado trabajo

We have so much work to do -> Nosotros tenemos tanto trabajo que hacer

Some: algunos/algunas - contable/incontable

Uso: se utiliza con contables e incontables en plural. Se utiliza en frases afirmativas e interrogativas para afirmar algo y expresar una cantidad indefinida, pero limitada

Ej frase afirmativa contable: She has some flowers -> Ella tiene algunas flores

Ej frase afirmativa incontable: There is some coffee in the kitchen -> Hay algo de café en la cocina

Ej frase interrogativa contable: Can I have some flowers? -> ¿Puedo coger algunas flores?

Ej frase interrogativa incontable: Would you like some coffee? -> ¿Quieres algo café?

Any: alguno/ninguno - contable/incontable

Uso: en los mismos caso que some pero en frases negativas e interrogativas

Ej frase negativa contable: She hasn't any flowers -> Ella no tiene ninguna flor

Ej frase negativa incontable: There isn't any coffee in the kitchen -> No hay ningun café en la cocina

Ej frase interrogativa contable: Are there any flowers? -> ¿Hay algunas flores?

Ej frase interrogativa incontable: Is there any coffee? -> ¿Hay algun café?

No, none: no, ninguno/a - contable/incontable

Uso: none se puede utilizar sin ir acompañado de un nombre contable o incontable, este va al final de la frase y como única respuesta a una pregunta que haga referencia a cantidades

Ej contable: How many books are there? None -> ¿Cuántos libros hay? Ninguno

Ej incontable: I have no money -> No tengo dinero

A lot of, lots of: muchos/as - contable/incontable

Uso: expresan grandes cantidades, se usa con contables e incontables. No se utiliza en frases interrogativas

Ej contable: He has a lot of books -> Tiene muchos libros

Ej incontable: I do not have a lot of money -> No tengo mucho dinero

Litte: poco/un poco - incontable

Uso: se utiliza para expresar cantidades pequeñas con incontables. A little significa una opinión positiva sobre la cantidad mientras que little implica una opinión negativa

Ej interrogativa: Do you have a little time? -> ¿Tienes un poco de tiempo?

Ej afirmativa: We have little time -> Tenemos poco tiempo

Few, a few: pocos/unos pocos - contable

Uso: se utiliza para expresar un número pequeño con contables. Few describe la cantidad de forma negativa y a few la describe de forma positiva

Ej interrogativa: Do you have a few coins? -> ¿Tienes unas pocas monedas?

Ej afirmativa: There are few tourists today -> Hoy hay pocos turistas
