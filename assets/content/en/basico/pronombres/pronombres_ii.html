Pronombres

Pronombres demostrativos

Pueden estar en singular o plural y que pueden hacer referencia a la distancia

This -> este/a/o Ej. singular y aquí: I like this book -> Me gusta este libro
That -> ese/a/o, aquél, aquello/a Ej. singular y allí: I like that book -> Me gusta ese libro
These -> estos/as Ej. plural y aquí: I like these libros -> Me gustan estos libros
Those -> esos/as, aquellos/as Ej. plural y allí: I like those libros -> Me gustan aquellos libros

Pueden ir acompañados de un nombre o pueden ir solos, por ejemplo:

This is a good book -> Éste es un buen libro
What is that? -> ¿Qué es eso?

Otros usos de los demostrativos

Podemos utilizar this para presentarmos. Ej. Hello. This is Mary -> Hola. Soy Maria
Para preguntar quién hay al otro lado utilizamos thst. Ej. Peter, is that you? -> ¿Eres tu, Peter?
Utilizamos this cuando presentamos a personas. Ej. Mery, this is my friend Lucy -> Maria, esta es mi amiga Lucy
Podemos utilizar that para referirnos a algo del pasado, acción reciente o pasada. Ej. That pizza was delicious -> Aquella pizza estaba deliciosa

Pronombres reflexivos

Se usan cuando el sujeto y el complemento del verbo son lo mismo, para ello se utilizan los pronombres reflexivos

Myself -> yo mismo, a mí Ej: I saw it myself -> Yo mismo lo vi
Yourself -> tú mismo, a tí Ej: Don't burn yourself! -> ¡No te quemes!
Himself -> él mismo, a sí mismo Ej: He hurt himself -> Se hizo daño.
Herself -> ella misma, a sí misma Ej: She did it -> Lo hizo ella misma
Itself -> él mismo, a sí mismo Ej: The cat scratched itself -> El gato se rascó
Ourselves -> nosotros mismos Ej: We made it ourselves -> Lo hemos hecho nosotros mismos
Yourselves -> vosotros mismos, ustedes mismos Ej: Did you paint the house yourselves? -> ¿Pintasteis la casa vosotros mismos?
Themselves -> ellos mismos Ej. They were speaking to themselves -> Ellos hablaban consigo mismos

Reglas gramaticales

Se pueden utilizar con la mayoría de verbos transitivos
Blame -> culpar, echar la culpa a
Cut -> cortar
Enjoy -> disfrutar
Help -> ayudar
Hurt -> hacerse daño
Introduce -> presentarse
Prepare -> preparar
Teach -> enseñar
Ej: How did he hurt himself? -> ¿Cómo se hizo daño?

Cuando queremos enfatizar el sujeto, se suele colocar el pronombre al final de la oración
Ej: We painted the house ourselves -> Pintamos la casa nosotros mismos

By + el pronombre reflexivo para indicar solo
Ej: She learned to read all by herself -> Ella aprendió a leer por sí misma

El uso del pronombre reflexivo con algunos verbos puede cambiar el significado
Ej: Help yourself to some coffee -> Sírvete un café tú mismo

No se utiliza para acciones que la gente suele hacer a sí misma
Ej: I have to shave myself every other day -> Tengo que afeitarme cada dos días

Se utilizan en algunas expresiones
Ej: Help yourself -> Sírvete tú mismo

Pronombres recíprocos

Each other A diferencia del reflexivo, tenemos dos sujetos diferentes que hablan el uno al otro
Ej: John and Peter speak to each other every day -> John y Peter se hablan cada día
