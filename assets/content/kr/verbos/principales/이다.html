이다 ~ Ser/Estar

이다 se utiliza para identificar personas, animales o cosas, equivale al verbo ser. En este caso el verbo no forma una palabra independiente, sino que se une al sustantivo que identifica
Conjugación del verbo según el nivel de formalidad
Raíz del verbo: 이- (la raíz se obtiene quitando 다) + ㅂ니다 / ㅂ니까
Sustantivo1는(은) sustantivo2 + verbo ser

Formal 입니다 oración enunciativa
입니까? oración interrogativa
se añade al sustantivo ya acabe en vocal o consonante
Formal STD si el sustantivo acaba en vocal: -예요
si el sustantivo acaba en consonante: -이에요
la oración interrogativa se hace con la entonación y el signo "?", igual que en español
Informal si el sustantivo acaba en vocal: -야
si el sustantivo acaba en consonante: -이야
la oración interrogativa se hace con la entonación y el signo "?", igual que en español

Vocabulario:

책 = libro
컴퓨터 = ordenador

Ejemplos:

  • Estilo formal
    • 책입니다 = Es un libro
    • 책입니까? = ¿Es un libro?
    • 컴퓨터입니다 = Es un ordenador
    • 컴퓨터입니까? = ¿Es un ordenador?
  • Estilo formal STD
    • 책이에요 = Es un libro
    • 책이에요? = ¿Es un libro?
    • 컴퓨터예요 = Es un ordenador
    • 컴퓨터예요? = ¿Es un ordenador?
  • Estilo informal
    • 책이야 = Es un libro
    • 책이야? = ¿Es un libro?
    • 컴퓨터야 = Es un ordenador
    • 컴퓨터야? = ¿Es un ordenador?
  • 저는 학생이에요 = Yo soy estudiante
  • 나는 배우야 = Yo soy actor/actriz
  • 오빠는 가수입니다 = Mi hermano mayor es cantante
