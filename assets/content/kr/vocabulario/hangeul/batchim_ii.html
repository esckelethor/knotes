Hangeul II: Batchim

Tipos de batchim II

Palatización

Las consonantes finales ㄷ / ㅌ se convierten en ㅈ / ㅊ y reemplazan a ㅇ en la siguiente sílaba

Tipo 1: ㄷ + 이 / 여

ㄷ + 이 / 여 => ㅈ se enlaza con la silaba siguiente => 지 / 져

Escrito: 해돋이
Leido: 해도지

Tipo 2: ㄷ + 히

ㄷ + 히 => pasa a ser 치

Escrito: 닫히다
Leido: 다치다

Tipo 3: ㄷ + 혀

ㄷ + 혀 => pasa a ser 쳐

Escrito: 묻혀서
Leido: 무쳐서

Tipo 4: ㅌ + 이 / 여

ㅌ + 이 / 여 => ㅊ y se enlaza con la silaba siguiente => 치 / 쳐

Escrito: 같이
Leido: 가치
Escrito: 붙여서
Leido: 부쳐서

Aspiración

Tipo 1:

Batchim ㄱ + ㅎ => batchim se enlaza con consonante siguiente inicial y se transforma en ㅋ k ingles. Vale para forma inversa -> batchim ㅎ + ㄱ

Tipo 2:

Batchim ㄷ / ㅅ + ㅎ => batchim se enlaza y se transforma ㅌ T ingles

Tipo 3:

Batchim ㅂ + ㅎ => batchim se enlaza y se transforma ㅍ P ingles

Tipo 4:

Batchim ㅈ + ㅎ => batchim se enlaza y se transforma ㅊ CH ingles. Vale para forma inversa -> batchim ㅎ + ㅈ

Escrito: 박하
Leido: 바카

Tensificación

Tipo 1:

ㅂ / ㄷ / ㄱ + ㅂ / ㄷ / ㄱ / ㅅ/ ㅈ => ㅂ / ㄷ / ㄱ+ ㅃ / ㄸ / ㄲ / ㅆ / ㅉ

Tipo 2:

ㄱ precedida de forma adverbial de futuro –(으)ㄹ => ㄱ -> ㄲ

Tipo 3:

Raíz de verbo terminado en ㄴ / ㅁ+ ㄷ / ㅈ / ㄱ estas se pronuncian ㄸ / ㅉ / ㄲ

Tipo 4:

Cuando un carácter chino (hanja) terminado en ㄹ le sigue ㄷ / ㅅ / ㅈ se pronuncia ㄸ / ㅆ / ㅉ

Tipo 5:

Ciertas palabras pueden tensificar a otras cuando se pronuncian juntas 가게 -> 까게

Ejemplos:
Escrito: 곧기
Leido: 곧끼
Escrito: 육박전
Leido: 육빡쩐 doble tensificación
