Hangeul II: Batchim

Tipos de batchim

Existen dos tipos de batchim: el batchim simples (de una sola letra), y el batchim compuesto (de dos letras o letras dobles); en concreto los correspondientes a las consonantes dobles ㄲ, ㄸ, ㅃ, ㅉ y ㅆ (que son las únicas dobles que pueden aparecer también como consonantes iniciales al principio de la sílaba)

  • ㅂ, ㅍ = P
  • ㄱ, ㄲ, ㅋ = K
  • ㄷ, ㅌ, ㅈ, ㅊ, ㅅ, ㅆ, ㅎ = T
  • ㄹ = L
  • ㅁ = M
  • ㄴ = N
  • ㅇ = NG

Cuando algunas consonantes finales interaccionan con ciertas consonantes iniciales de la siguiente sílaba, cada una de ellas o ambas, cambian de sonido

  • Enlace (continuidad)
  • Nasalización (cambio a sonidos nasales)
  • Palatalización (cambio a sonidos palatales)
  • Aspiración
  • Tensificación
  • Liquidación (cambio a sonidos líquidos)
  • Otros fenómenos fonéticos relacionados con ㄹ

Enlace o liason

La consonante final se convierte en inicial si la inicial es ㅇ

Escritura Lectura
금요일 그묘일
없어요 업써요 *Excepción

Tip!

Excepción:

  • Cuando ㅅ es consonante final seguida por ㅇ, se produce enlace y el sonido cambia a ㅆ
  • ㅇ porque no se traslada el sonido a NG al comienzo de la silaba
  • ㅎ se silencia cuando sigue a una silaba de vocal inicial -> se llama ELIMINACION

Nasalización:

Los sonidos nasales se corresponden a las consonantes: ㄴ, ㅁ, ㅇ (sonido ng)

Tipo 1: ㅂ / ㅍ + ㄴ / ㅁ => ㅁ + ㄴ / ㅁ

Cuando una consonante final ㅂ / ㅍ precede a una consonante inicial ㄴ / ㅁ el sonido de la consonante final es ㅁ

Palatización

Las consonantes finales ㄷ / ㅌ se convierten en ㅈ / ㅊ y reemplazan a ㅇ en la siguiente sílaba

Tipo 1: ㄷ + 이 / 여

ㄷ + 이 / 여 => ㅈ se enlaza con la silaba siguiente => 지 / 져

ㄷ + 히 => pasa a ser 치

Tipo 3: ㄷ + 혀

ㄷ + 혀 => pasa a ser 쳐

Tipo 4: ㅌ + 이 / 여

Aspiración

Tipo 1:

Batchim ㄱ + ㅎ => batchim se enlaza con consonante siguiente inicial y se transforma en ㅋ k ingles. Vale para forma inversa -> batchim ㅎ + ㄱ

Tipo 2:

Batchim ㄷ / ㅅ + ㅎ => batchim se enlaza y se transforma ㅌ t ingles

Tipo 3:

Batchim ㅂ + ㅎ => batchim se enlaza y se transforma ㅍ p ingles

Tipo 4:

Batchim ㅈ + ㅎ => batchim se enlaza y se transforma ㅊ ch ingles. Vale para forma inversa -> batchim ㅎ + ㅈ

Escritura Lectura
박하 바카

Tensificación

Tipo 1:

ㅂ / ㄷ / ㄱ + ㅂ / ㄷ / ㄱ / ㅅ / ㅈ => ㅂ / ㄷ / ㄱ + ㅃ / ㄸ / ㄲ / ㅆ / ㅉ

Tipo 2:

ㄱ precedida de forma adverbial de futuro –(으)ㄹ => ㄱ -> ㄲ

Tipo 3:

Raíz de verbo terminado en ㄴ / ㅁ + ㄷ / ㅈ / ㄱ estas se pronuncian ㄸ / ㅉ / ㄲ

Tipo 4:

Cuando un carácter chino (hanja) terminado en ㄹ le sigue ㄷ / ㅅ / ㅈ se pronuncia ㄸ / ㅆ / ㅉ

Tipo 5:

Ciertas palabras pueden tensificar a otras cuando se pronuncian juntas 가게 -> 까게

Otros fenómenos fonéticos relacionados con ㄹ

Existen determinadas situaciones dentro de una palabra en las que ㄹ cambia a ㄴ al estar precedida por otras consonantes:

Tipo 1

ㅇ + ㄹ = ㅇ + ㄴ -> Si ㄹ está precedida de ㅇ, ㄹ cambia a ㄴ. ㅇ no varía

Escritura Lectura
정류장 정뉴장
장래 장내

Tipo 2

ㅂ + ㄹ = ㅁ + ㄴ -> Si ㄹ está precedida de ㅂ, ㄹ cambia a ㄴ. ㅂ cambia a ㅁ

Escritura Lectura
입력 임녁
엽록체 염녹체

Tipo 3

ㄱ + ㄹ = ㅇ + ㄴ -> Si ㄹ está precedida de ㄱ, ㄹ cambia a ㄴ. ㄱ cambia a ㅇ

Escritura Lectura
박람회 방남회
악령 앙녕
