Como decir a hora

Para decir la hora se utilizan los 2 sistemas numéricos:

  • Para la hora se usa el sistema sistema nativo-coreano seguido de 시
  • Para los minutos se usa el sistema sino-coreano seguido de 분

Los números nativo-coreanos del 1 al 4 cambian ligeramente al añadir un sufijo contable (시)
Para indicar y media se puede usar 30 minutos: 삼십분 o 반
Para diferenciar AM y PM se usan estas palabras que equivalen a:

  • 오전 = Antes del mediodía AM
  • 오후 = Después del mediodía PM
  • 저녁 = Por la tarde PM
  • 밤 = Por la noche PM

Construcción

NºNativo-coreano 시 NºSino-coreano 분

Ejemplos

  • 01:01 = 한시 일분
  • 02:10 = 두시 십분
  • 06:23 = 여섯시 이십삼분
  • 12:45 = 열두시 사십오분 Aunque sea 12 al igual que con el 2 también se transforma por ser decenas
  • 21:30 = 스물한시 삼십분 / 스물한시 반 Aunque sea 21 al igual que con el 2 también se transforma por ser decenas
  • 01:15 PM = 오후 한시 십오분
  • 07:35 PM = 밤 일곱시 삼십오분 En corea se considera noche a partir de las 7 ya que es a la hora que suelen cenar
  • 06:30 AM = 오전 여섯시 삼십분 / 오전 여섯시 반
